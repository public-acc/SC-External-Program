# 프로그램 저장 방식 및 이용 안내

본 문서는 프로그램의 이용 목적, 저장 방식, 데이터 처리 범위, 책임 범위를 사용자에게 안내하기 위한 문서입니다.

---

## 1. 프로그램 안내

본 프로그램은 스타크래프트 유즈맵을 보다 편리하고 즐겁게 플레이할 수 있도록 제작·배포되는 무료 보조 프로그램입니다.

본 프로그램은 유즈맵의 세이브/로드 입력 과정을 보조하기 위한 도구이며, 특정 유즈맵의 운영, 후원, 보상, 확률, 업데이트, 랭킹, 저장소 운영을 보증하거나 관리하지 않습니다.

각 유즈맵의 플레이, 후원, 보상, 운영, 업데이트, 분쟁은 해당 유즈맵 제작자와 사용자 사이의 관계입니다.

---

## 2. 프로그램 제작자의 역할

프로그램 제작자의 역할은 다음 범위로 제한됩니다.

```txt
- 유즈맵 세이브/로드 입력 보조
- 로컬 저장 파일 생성 및 읽기
- 유즈맵 제작자가 설정한 저장 방식에 따른 데이터 처리 보조
- 프로그램 프로그램 자체의 정상 동작 유지
```

프로그램 제작자는 다음 사항에 관여하지 않습니다.

```txt
- 유즈맵 후원 또는 결제 중개
- 유즈맵 보상 지급 또는 보증
- 유즈맵 확률, 밸런스, 업데이트 관리
- 유즈맵 제작자의 저장소 운영
- 유즈맵 제작자의 서버 또는 스프레드시트 관리
- 유즈맵 이용자와 제작자 사이의 분쟁 조정
- 특정 유즈맵의 수익 구조 또는 운영 정책 보증
```

---

## 3. 악용 방지를 위한 설계 방향

본 프로그램은 유즈맵 제작자가 불순한 목적으로 악용하기 어렵도록 다음과 같은 방향을 고려하여 설계됩니다.

```txt
- 기본 저장 방식은 로컬 저장 중심으로 구성
- 유즈맵별 저장 데이터 분리
- 제작자별 설정 분리
- 저장 데이터의 임의 변조 방지 구조 적용
- 제작자 키와 프로그램 키의 역할 분리
- 프로그램 제작자가 유즈맵 제작자의 개별 키를 알 수 없는 구조 지원
```

다만 프로그램 제작자는 모든 유즈맵 제작자의 운영 방식, 후원 방식, 보상 지급, 업데이트 지속 여부를 사전에 확인하거나 보증할 수 없습니다.

따라서 특정 유즈맵의 운영 방식, 후원, 보상, 확률, 업데이트, 저장 데이터 복구와 관련된 사항은 해당 유즈맵 제작자에게 확인해야 합니다.

---

## 4. 로컬 저장 방식

저장소를 활용하는 유즈맵의 경우, 기본적으로 저장 데이터는 사용자 PC의 다음 폴더에 저장됩니다.

```txt
./data/
```

로컬 저장 방식에서는 저장 데이터가 사용자 PC 내부에 저장되며, 프로그램 제작자의 서버로 전송되거나 보관되지 않습니다.

사용자는 `./data/` 폴더의 파일을 직접 백업할 수 있습니다.

저장 파일이 손상되거나 삭제될 경우 세이브/로드에 문제가 발생할 수 있으므로, 중요한 저장 데이터는 사용자가 직접 백업하는 것을 권장합니다.

---

## 5. 로컬 저장 데이터와 복호화

저장 데이터는 유즈맵별 설정과 키 구조에 따라 암호화 또는 난독화될 수 있습니다.

유즈맵 제작자가 별도의 임의 키를 부여하고, 해당 키가 프로그램 제작자에게 제공되지 않은 구조에서는 프로그램 제작자도 저장 파일의 실제 내용을 복호화할 수 없습니다.

즉, 다음 조건에서는 프로그램 제작자가 저장 데이터의 실제 값을 확인하거나 복구할 수 없습니다.

```txt
- 유즈맵 제작자가 별도 키를 사용하는 경우
- 해당 키가 프로그램 제작자에게 제공되지 않은 경우
- 저장 파일이 손상되었거나 일부 내용이 유실된 경우
- 사용자가 저장 파일을 임의로 수정한 경우
```

저장 파일의 손상, 삭제, 분실, 임의 수정으로 인해 발생하는 불이익은 사용자 본인에게 있을 수 있습니다.

---

## 6. Apps Script / 스프레드시트 저장 기능

일부 유즈맵은 제작자의 설정에 따라 Google Apps Script 또는 외부 스크립트를 통해 스프레드시트에 저장 데이터를 기록할 수 있습니다.

이 경우 데이터 흐름은 다음과 같습니다.

```txt
사용자 PC의 프로그램
  → 유즈맵 제작자가 지정한 Apps Script 또는 외부 스크립트
  → 유즈맵 제작자가 관리하는 스프레드시트
```

프로그램 제작자는 해당 Apps Script, 스프레드시트, 저장 데이터, 접근 권한, 복구, 삭제, 백업, 보안 설정을 운영하거나 관리하지 않습니다.

Apps Script 또는 스프레드시트 저장 기능을 사용하는 유즈맵의 경우, 저장소 운영 책임은 해당 유즈맵 제작자에게 있습니다.

---

## 7. 전송될 수 있는 데이터

Apps Script 또는 스프레드시트 저장 기능을 사용하는 경우, 유즈맵 제작자의 설정에 따라 다음 정보가 전송될 수 있습니다.

```txt
- 유즈맵 식별 정보
- 사용자 닉네임 또는 식별용 ID
- 저장 슬롯 번호
- 저장 코드 또는 저장 데이터
- 저장 시각
- 맵 버전 또는 프로그램 버전
```

전송되는 데이터의 구체적인 항목은 유즈맵 제작자의 설정에 따라 달라질 수 있습니다.

결제 정보, 비밀번호, 주민등록번호, 금융 정보 등은 본 프로그램의 세이브/로드 기능에 필요하지 않으며, 프로그램는 이러한 정보를 요구하지 않습니다.

---

## 8. 보안 및 바이러스 검사 안내

공식 배포처가 아닌 경로에서 받은 파일은 변조되거나 바이러스에 감염되었을 수 있습니다.

프로그램 실행 전, 사용자는 아래와 같은 방법으로 파일의 안전성을 확인할 수 있습니다.

```txt
- 공식 배포처에서 다운로드했는지 확인
- 파일명이 임의로 변경되지 않았는지 확인
- 압축 파일 내부에 알 수 없는 실행 파일이 추가되어 있지 않은지 확인
- 백신 프로그램 또는 VirusTotal을 통한 검사
```

VirusTotal 검사 페이지:

```txt
https://www.virustotal.com/gui/home/upload
```

프로그램 제작자가 배포한 원본 파일이 아닌 변조 파일, 재배포 파일, 비공식 공유 파일을 실행하여 발생한 문제는 프로그램 제작자가 확인하거나 책임지기 어렵습니다.

---

## 9. 문의 범위

프로그램 제작자에게 문의할 수 있는 사항은 다음과 같습니다.

```txt
- 프로그램 실행 오류
- 로컬 저장/로드 기능 오류
- 프로그램 UI 또는 기능 오류
- 프로그램 자체의 보안 문제
```

다음 사항은 해당 유즈맵 제작자에게 문의해야 합니다.

```txt
- 유즈맵 후원 관련 문제
- 보상 미지급 문제
- 확률 또는 밸런스 문제
- 유즈맵 업데이트 중단
- Apps Script 또는 스프레드시트 저장소 문제
- 저장 데이터 복구 요청
- 유즈맵 운영 분쟁
```

본 프로그램은 유즈맵 제작자가 설정한 저장 방식을 실행하는 클라이언트 도구입니다.
프로그램 제작자는 프로그램 프로그램 자체의 안전성과 정상 동작을 위해 노력하지만, 유즈맵 제작자가 운영하는 서버, 스프레드시트, 보상 정책, 후원 정책, 운영 정책을 관리하거나 보증하지 않습니다.
